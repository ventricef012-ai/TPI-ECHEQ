# TPI-ECHEQ
## Sistema de Pago con Cheques Electrónicos (Echeq)

### Descripción
Simulador del proceso administrativo de pago con cheques electrónicos (Echeq) 
desarrollado para el Trabajo Práctico Integrador de Organización Empresarial - UTN TUPaD.

### Requisitos
- Python 3.x instalado

### Cómo ejecutar
1. Clonar el repositorio
2. Abrir una terminal en la carpeta del proyecto
3. Ejecutar el siguiente comando:

``` python Bot_echeq.py

### Archivos
- `Bot_echeq.py` — Código principal del simulador
- `facturas.csv` — Base de datos simulada de facturas

### Flujo del proceso
1. Ingreso y autorización de facturas (Compras)
2. Recepción y control en Tesorería
3. Firma y emisión de echeqs (Firmante)
4. Envío automático de comprobantes al proveedor

### Tecnologías utilizadas
- Python 3
- Módulo csv (base de datos simulada)
